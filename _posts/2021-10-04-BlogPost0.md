### Construct a visualization

A simple direction of investigation could be the body mass distribution for each species: Do the penguines from all three species roughly have the same body mass?

In order to answer the above the question, we can make a box-whisker plot to help us visualize the data set.


```python
# create a single figure with a single axis (1 row x 1 column)
# and stretch them out to a better aspect ratio 8,6
fig,ax = plt.subplots(1,figsize= (8,6))

# draw a box plot showing the distribution of body masses with respect to a penguin's species
# set penguins to be the dataset for plotting, x to be body mass, and y be species
# set the width to be 0.3 for better aspect ratio of the boxes
sns.boxplot(data = penguins, y = "Species", x = "Body Mass (g)", width = 0.3, dodge = False)

# add a title to the graph
plt.title('Body Mass Distribution for Each Species')
```
